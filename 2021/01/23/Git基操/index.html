2. 查看当前仓库状态 — git status

3. 添加更改（追踪） — git add .

4. 查看更改内容 — git diff

5. 撤回追踪 — git reset

6. 提交更改 — git commit

7. 设置一些不提交的文件 — 创建.gitignore

8. 删除已经存在的追踪
